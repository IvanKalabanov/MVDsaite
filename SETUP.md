# Инструкция по настройке и запуску

## Быстрый старт

```bash
npm install
npm run dev
```

После запуска откройте `http://localhost:5173`.

## Учётные записи по умолчанию

| Роль | Логин | Пароль |
| --- | --- | --- |
| Администратор | `admin` | `admin123` |
| Руководитель | `leader` | `leader123` |
| Сотрудник | `employee` | `employee123` |
| Гражданин | `user` | `user123` |

## Где хранятся данные

- Данные сохраняются в `localStorage` браузера (ключ `mvdsai-local-data`).
- При первом запуске подтягиваются моковые наборы.
- Очистить всё можно через консоль браузера:

```js
localStorage.removeItem('mvdsai-local-data');
window.location.reload();
```

## Возможности администратора

Администратор имеет полный доступ к:

1. Панели статистики.
2. Управлению пользователями и ролями.
3. Управлению базой данных нарушителей.
4. Управлению контентом (новости, сотрудники, руководство).
5. Настройкам интерфейса.

## Добавление данных

Все операции выполняются через UI:

- Новости — раздел «Новости».
- Заявления — раздел «Подача заявлений».
- База данных — раздел «База данных».
- Сотрудники — раздел «Сотрудники».
- Руководство — раздел «Руководство».

## Решение проблем

- **Не видно новых данных** — проверьте консоль браузера, возможно, localStorage переполнен.
- **Нужно начать заново** — очистите `mvdsai-local-data`, как описано выше.
- **Хочется поделиться данными** — экспортируйте значение `localStorage.mvdsai-local-data` и импортируйте его на другом устройстве.

Никаких дополнительных серверов, портов и баз данных поднимать не требуется.
